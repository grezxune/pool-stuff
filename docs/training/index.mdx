## Our goal

The goal of Pool Stuff is to be a useful resource for you no matter your skill level. For that reason, I am going to start from the beginning. If you need to find something specific, hopefully this site is organized in such a way which makes that easy. If it isn't, open a PR with an update or message me with suggestions to improve it.

## Why are you here?

1. I want to learn the game
2. I want to improve my game
3. I am looking for rules
3. I am looking to find leagues / tournaments
4. I want to contribute to this resource
